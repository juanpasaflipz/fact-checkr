## üéØ Conversion Strategy

### Free-to-Pro Conversion Triggers

#### 1. **Soft Paywall Triggers** (Non-Blocking)
- Show upgrade prompt after 8th verification attempt (with clear value message)
- Display "Pro Feature" badge on advanced analytics
- Show usage counter: "7/10 verifications used this month"
- Highlight time-saving features: "Save 2 hours/week with bulk verification"

#### 2. **Hard Paywall Triggers** (Blocking)
- Block export functionality
- Block historical data beyond 7 days
- Block API access
- Limit search to 50 queries/day

#### 3. **Value-Driven Prompts**
- After viewing analytics: "Unlock 1 year of trends with Pro"
- After search: "Save this search as an alert with Pro"
- After viewing trends: "Export this data with Pro"

#### 4. **Trial Period** (Optional)
- Offer **7-day free Pro trial** (credit card required)
- No auto-renewal during trial
- Cancel anytime during trial = free forever
- After trial: convert to paid or downgrade to Free
